# guide-mainsite-aiyouxi

本仓库用于归档和发布多个独立 HTML 页面，作为项目资源集合。

## 项目简介

这是一个以 HTML 页面为主要内容的归档仓库，收录了若干独立开发的静态页面。页面内容覆盖不同主题，适合直接通过浏览器访问查看。仓库不依赖特定后端服务，所有页面均为前端静态实现。

## 目录说明

```
guide-mainsite-aiyouxi/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 页面引用的资源文件（如图片、样式、脚本）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面，可直接访问。
- `assets/`：放置页面所需的共用资源，按类型组织子文件夹。
- 根目录的 `index.html` 可作为导航入口，方便浏览全部页面。

## 页面归档说明

- 所有页面均为纯静态 HTML，无后端依赖。
- 页面间相互独立，可单独打开浏览。
- 部分页面可能引用外部资源（如字体库、图标库），需联网加载。
- 页面内容可能随仓库更新而调整，请以最新提交为准。

## 维护说明

- 欢迎提交 Issue 反馈页面问题或提出改进建议。
- Pull Request 请确保新增页面不包含恶意代码或外部链接跳转。
- 不建议将本仓库用于商业推广或诱导点击行为。
- 仓库维护者有权合并或拒绝不符合归档目的的提交。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/guide-mainsite-aiyouxi.git
   ```
2. 直接在浏览器中打开 `index.html` 或 `pages/` 下的任意 HTML 文件。
3. 也可通过 GitHub Pages 等静态托管服务在线访问（需自行配置）。
